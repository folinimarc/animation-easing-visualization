Animation Easing Visualization

Easing Functions describe the relationship between a normalized time value (x-axis) and a normalized animation progress value (y-axis). The normalized time value will move steadily from 0 to 1, with 0 and 1 indicating the start and end time of the animation. The normalized animation progress ranges from 0 to 1 and indicates how far along the animation is relative to the start and end states. These states can any property that can be interpolated over time, such as color, position, scale, opacity and many more.
